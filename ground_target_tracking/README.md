# ground_target_tracking

This is the application package. The project documentation — requirements,
versions, run instructions, assignment mapping, and limitations — lives in the
repository-root [README](../README.md) (the page GitHub renders).

Quick start, from the repository root:

```bash
python3 -m pip install -r ground_target_tracking/requirements.txt
python3 -m ground_target_tracking.main
```

> The package is named `ground_target_tracking` for import stability; the tracker
> itself is general-purpose — it follows any clear user-selected object.
